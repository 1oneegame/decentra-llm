# welcome to the yerayal team's solution!

## quick start

### backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python run_server.py
````

### frontend

```bash
cd frontend
npm install
npm run dev
```
