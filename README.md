# Welcome to the yerayal team's solution!

## Get started quick:

1. ```cd backend```
   1.1. ```python -m venv venv```
   1.2. ```source venv/bin/activate```
   1.3. ```pip install -r requirements.txt```
   1.4. ```python run_server.py```
2. ```cd frontend```
   2.1. ```npm i```
   2.2. ```npm run dev```
